Kebakaran Pasar Rawa Kalong Bekasi, Ratusan Kios Terbakar
Ratusan kios di Pasar Rawa Kalong, Bekasi, terbakar. Padatnya permukiman penduduk menyulitkan petugas pemadam di lokasi.
 Bekasi - Ratusan kios di Pasar Rawa Kalong, Bekasi, terbakar. Padatnya permukiman penduduk menyulitkan petugas pemadam di lokasi."Masih terbakar, merah menyala. Diperkirakan ratusan kios sudah terbakar," kata pembaca detikcom Jamal saat dihubungi, Jumat (12/1/2018).Jamal menjelaskan lokasi pasar yang berada di Jalan Bumi Sani Permai, Gang Tanggul, Bekasi, itu juga menyulitkan petugas damkar. Bangunan kios yang mudah terbakar juga mempercepat api merembet ke sekitarnya. "Lokasinya di dalam gang, jadi damkar susah masuknya. Padat permukiman. Selain itu, kios-kiosnya terbuat dari kayu dan gipsum, jadi mudah terbakar," jelasnya.Api masih menyala di Pasar Rawa Kalong, Bekasi. (Jamal/pembaca detikcom)Ditambahkan Jamal, warga sempat mendengar ledakan sebelum api menyambar. "Warga bilang ada ledakan, terus api keluar dan merembet ke kios lainnya," lanjutnya.Hingga kini, petugas pemadam kebakaran masih berusaha memadamkan api. Tiga unit mobil pemadam kebakaran dari Pos Babelan dan Tambun Selatan sudah berada di lokasi. (adf/nvl)
