Putra Trump Sebut Kebakaran di Trump Tower dari Menara Pendingin
Kebakaran melanda Trump Tower. Menurut putra Trump, tadinya ada api kecil di atap gedung.
 New York - Petugas Pemadam Kebakaran masih berada di Trump Tower untuk memadamkan api. Putra Donald Trump, Eric trump, menyebut api berasal dari pendingin yang berada di bagian atas gedung. "Ada api listrik kecil di menara pendingin di atap Trump Tower," kata Eric Trump melalui akun Twitternya @EricTrump seperti dikutip detikcom, Senin (8/1/2018).There was a small electrical fire in a cooling tower on the roof of Trump Tower.
