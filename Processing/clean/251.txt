Kebakaran Hanguskan Dua Warung dan Satu Rumah di Lamongan
Dua warung dan satu rumah di Lamongan ludes dilalap api. Meski tak ada korban jiwa dalam kejadian ini, namun kerugian ditaksir mencapai jutaan rupiah.
 Lamongan - Dua warung dan satu rumah di Lamongan ludes dilalap api. Meski tak ada korban jiwa dalam kejadian ini, namun kerugian ditaksir mencapai jutaan rupiah.DariiInformasi yang dihimpun detikcom, kebakaran yang membuat dua warung dan satu rumah habis terbakar tersebut terjadi tak jauh dari Pertigaan traffic light Sumlaran, tepatnya di Terminal MPU yang berada di Desa/ Kecamatan Sukodadi. Saat kejadian, dua warung tersebut sedang tutup. Salah seorang saksi mata, Sadam Husein kepada wartawan menuturkan, kejadian kebakaran ini bermula dari warung milik Samsiah yang saat itu sudah dalam keadaan tutup karena pemiliknya pulang. Saat itu, kata Sadam, kemungkinan Samsiah sedang meninggalkan warung dalam keadaan tungku api yang masih menyala. "Sebelum api membesar, saya sempat mencium bau hangus dan saya lihat keluar ternyata api sudah membakar warung milik Bu Samsiah," kata Sadam di lokasi kejadian.Setelah menyebar, kata Sadam, api kemungkinan juga menyambar tabung elpiji yang ada di dalam warung karena sempat terdengar ledakan. "Melihat api tersebut, saya langsung lari dan teriak minta tolong, namun api semakin membesar karena semua bangunan yang terbakar terbuat dari kayu yang mudah terbakar," terang Sadam.Setelah menghanguskan warung milik Samsiah, si jago merah kemudian menyambar warung milik Suratno, yang berada di sebelahnya yang kebetulan juga sedang tutup. Selain itu, api juga menyambar rumah Kartini yang tak lain adalah ibu dari Sadam. "Kejadiannya tiba-tiba, ada asap dari utara dan ternyata api langsung membesar ke selatan hingga ke rumah dan warung," ujar sadam.Meski tak menimbulkan korban jiwa, namun mengamuknya si jago merah ini menyebabkan kerugian jutaan rupiah. Sementara, petugas pemadam kebakaran yang datang ke lokasi kebakaran langsung mencoba memadamkan api dan melakukan pembasahan setelah api berhasil dipadamkan. Petugas kepolisian pun hingga kini masih menyelidiki penyebab pasti kebakaran ini. (iwd/iwd)
