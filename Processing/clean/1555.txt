5 Kios dan 2 Rumah di Obyek Wisata Sarangan Terbakar
Kebakaran terjadi di obyek wisata Telaga Sarangan Magetan. Lima kios serta dua rumah ludes terbakar.
 Magetan - Kebakaran terjadi di obyek wisata Telaga Sarangan Magetan. Lima kios serta dua rumah ludes terbakar."Iya betul, ada kebakaran di Sarangan. Lima kios dan dua rumah terbakar," ujar Kasi Kedaruratan dan Logistik BPBD Magetan Fery Yoga Shaputra saat di hubungi detikcom, Jumat (19/1/2018).Kebakaran, kata Fery, diduga terjadi dari percikan api di salah satu dapur rumah makan milik Ibu Mur. Namun untuk penyebab pasti masih dalam proses penyelidikan. "Ini masih dugaan, api berasal dari dapur salah satu depot. Tapi sementara ini masih dalam proses identifikasi," ungkap Fery.Fery menambahkan, proses pemadaman tidak mengalami kesulitan. Awalnya warga berusaha memadamkan api dengan alat seadanya sambil menunggu datangnya PMK. Begitu PMK datang, api dengan cepat segera dikuasai.Tidak ada korban jiwa dalam musibah ini. Nilai kerugian masih dilakukan penghitungan."Api sudah padam satu jam setelah terbakar. Untuk kerugian masih dihitung," pungkas Fery. (iwd/iwd)
