Kebakaran di Pasar Rawa Kalong Bekasi, 3 Unit Damkar Dikerahkan
Kebakaran terjadi di Pasar Rawa Kalong, Bekasi. Tiga unit mobil pemadam kebakaran dikerahkan.
 Bekasi - Kebakaran terjadi di Pasar Rawa Kalong, Bekasi. Tiga unit mobil pemadam kebakaran dikerahkan."Benar, kebakaran di Pasar Rawa Kalong. Untuk sementara dua unit dari Pos Babelan dan satu unit dari Tambun Selatan menuju lokasi," kata petugas piket Pemadam Kebakaran Kabupaten Bekasi, Abdul Kodir, saat dimintai konfirmasi, Jumat (12/1/2018).Abdul menjelaskan laporan kebakaran diterimanya sekitar pukul 20.30 WIB. Api diperkirakan berasal dari beberapa kios yang ada di lokasi kejadian. "Informasi di lapangan, api berasal dari beberapa kios yang ada di sana. Masih dalam penanganan," jelasnya.Belum diketahui penyebab kebakaran maupun korban jiwa dalam kebakaran itu. Petugas pemadam kebakaran masih berusaha mengendalikan api. (adf/nvl)
