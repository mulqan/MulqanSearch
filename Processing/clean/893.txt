Pabrik Tahu Meledak di Makassar, Warga Panik
Pabrik tahu di Makassar meledak. Api dengan cepat menghabiskan pabrik tahu yang ada di Jalan Kemauan Makassar.
 Makassar - Pabrik tahu di Makassar meledak. Api dengan cepat menghabiskan pabrik tahu yang ada di Jalan Kemauan Kelurahan Maccini Parang Kecamatan Makassar.Peristiwa kebakaran ini terjadi sekitar pukul 16.30 Wita. Warga mengaku sempat mendengar ledakan dari arah pabrik tahu"Ada sempat ledakan dari arah pabrik tahu, tiba tiba api muncul dan membakar," kata Japa kepada wartawan di lokasi, Jumat (12/1/2018). Tim pemadam kebakaran Kota Makassar harus mengerahkan 25 unit mobil damkar untuk memadamkan api. Meski berada di gang sempit api segera dapat di kendalikan."Kalau dari laporan warga sekitar jam 5 sore tadi. Kami terjungkan 25 unit damkar dari semua posko untuk memadamkan api, alhamdullilah terkendali," jelas Danton Damkar Makassar Saharuddin.Hingga kini, petugas kepolisian masih menyelidiki penyebab kebakaran yang menghaguskan pabrik tahu. Kuat dugaan akibat ledakan saat pembakaran pembuatan tahu. (asp/asp)
