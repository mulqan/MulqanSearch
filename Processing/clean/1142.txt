Mobil Sedan Terbakar Hebat di Area Parkir Merapi Park Sleman
Sebuah mobil jenis sedan ludes terbakar api di tempat parkir obyek wisata Merapi Park, Sleman siang ini. Tak ada korban jiwa dalam peristiwa ini.
 Sleman - Sebuah mobil jenis sedan ludes terbakar api di tempat parkir obyek wisata Merapi Park, Sleman siang ini. Tak ada korban jiwa dalam peristiwa ini. "Tadi wisatawan mau pulang dari Merapi Park, pas nyalain mesin, tiba-tiba mobilnya bunyi letupan, terus keluar asap dan api dari bawah mobil," kata seorang wisatawan, Eko Murniasih, di lokasi Desa Hargobinangun, Kecamatan Pakem, Sleman, Senin (1/1/2018).Peristiwa kebakaran terjadi sekitar pukul 14.15 WIB. Saat itu, Murni langsung berteriak kepada pengemudi agar keluar dari mobil. Sekitar 4 orang yang di dalam mobil langsung keluar. "Penumpang dan sopir selamat, bisa keluar," jelasnya.Seorang saksi lain, Muhammad Nasir melihat ada percikan api di bawah kap mesin mobil. Dia dan wisatawan yang mengetahui langsung berupaya memadamkan api dengan menyiram memakai air bekas cucian warung di sekitar tempat parkir."Tadi langsung disiram pakai air ember, galon, tapi api tetap membesar," jelasnya. Hingga kini api belum bisa dipadamkan. Terlihat asap hitam tebal masih membumbung tinggi. Belum diketahui persis penyebab terbakarnya mobil tersebut. (sip/sip)
