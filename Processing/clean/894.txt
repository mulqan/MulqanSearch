Stasiun Nottingham Inggris Kebakaran, Api Berawal dari Toilet
Stasiun kereta Notthingham, Inggris, mengalami kebakaran. Sebanyak 60 petugas pemadam kebakaran dikerahkan untuk memadamkan api yang masih menyala.
 Nottingham - Stasiun kereta Notthingham, Inggris, mengalami kebakaran. Sebanyak 60 petugas pemadam kebakaran dikerahkan untuk memadamkan api yang masih menyala. Dilansir dari media Inggris, Sky News, Jumat (12/1/2018), kebakaran mulai muncul sejak Jumat pagi waktu setempat. Hingga siang hari waktu London, api dilaporkan belum berhasil dipadamkan. Bahkan diketahui api telah menyebar hingga ke bagian atap stasiun. Petugas pemadam tiba di stasiun tersebut sekitar pukul 06.30 pagi. Komandan Dinas Penyelamat dan Pemadam Kebakaran Nottinghamshire, Bryn Coleman, menyatakan proses pemadaman masih terus berlangsung. Petugas pemadam menggunakan alat bantu pernapasan saat bekerja. Menurut Coleman, api saat ini juga telah mencapai kawasan jalan penghubung antara gedung stasiun lama dan baru. Coleman menjelaskan, laporan awal yang diterima menyebut kebakaran berawal dari salah satu toilet stasiun. Api kemudian menyebar ke bagian stasiun lainnya. "Konstruksinya cukup kompleks. Kami memiliki 12 unit alat pemadam kebakaran di sini, ditambah peralatan pemadam kebakaran khusus," ujar Coleman. Para penumpang maupun pegawai stasiun telah berhasil dievakuasi seluruhnya. Belum ada laporan korban jiwa akibat peristiwa ini. Penyebab kebakaran masih belum bisa diketahui. Polisi menyatakan lalu lintas di sekitar lokasi kebakaran mengalami gangguan cukup parah. Sejumlah ruas jalan seperti Station Street, Carrington Street dand Queen's Road ditutup untuk sementara. Operator layanan kereta East Midlands memperkirakan operasional kereta di wilayahnya akan mengalami gangguan nyaris di semua rute sepanjang Jumat (12/1) pagi waktu setempat.
